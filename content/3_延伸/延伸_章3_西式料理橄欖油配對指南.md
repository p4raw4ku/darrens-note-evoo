# 延伸：西式料理橄欖油配對指南

> 從 [[章3_烹飪應用|章 3.2.7]] 拆出的深度版。章 3.2.7 留「7 國配對速查表」短段，這份是義法西希臘葡北非中東 7 個菜系的橄欖油應用完整展開。

---

## 引言

西式料理裡 EVOO 不只是「拿來煎」的工具——**它常是料理的核心風味組件**，與食材並列。下面分國別／菜系深度展開。

---

## 義式（Italian）

- **Aglio e olio**（蒜油義大利麵）：油與蒜本身就是醬汁——必須用優質 EVOO（中強度、Toscano IGP 或 Frantoio 單一品種）；用差油這道菜直接崩塌
- **Pesto alla Genovese**：傳統石臼搗製，配方為羅勒、松子、大蒜、Parmigiano + Pecorino、**Ligurian Taggiasca EVOO**（風味溫和、不蓋過羅勒）；用 Picual 等強苦辛 EVOO 會與松子的甜衝突
- **Caprese**：蕃茄 + Mozzarella + 羅勒 + EVOO；finishing 用 Toscano 或 Umbria DOP 中強度
- **Bruschetta**：烤麵包 + 蒜 + 蕃茄 + 高品質 EVOO 大量淋；經典「油本身是主角」
- **Bagna càuda**（Piemonte 沾醬）：EVOO + 鯷魚 + 大蒜長時間慢燉 → 蔬菜沾醬；用溫和 EVOO 與奶油可選擇混合
- **Tomato sauce（Sugo）**：開頭用 EVOO 炒 soffritto（洋蔥 + 西芹 + 紅蘿蔔）；最後常再淋生 EVOO（章 3.5.2 finishing 用法）

---

## 法式（French）

法式傳統以**奶油為主、EVOO 為輔**，但南法（Provence、Côte d'Azur）大量使用 EVOO：

- **Vinaigrette**（油醋醬）：標準比例 3:1（油:醋）；用中強度 EVOO + Dijon 芥末 + 紅酒醋
- **Salade Niçoise**：地中海鮪魚沙拉、EVOO 淋；Provence 風味
- **Pissaladière**（南法洋蔥派）：類似 pizza 餅、EVOO 烤洋蔥 + 鯷魚 + 黑橄欖
- **Tapenade**：橄欖泥醬，黑橄欖 / 綠橄欖 + 鯷魚 + 酸豆 + EVOO 打成糊；麵包、烤魚的醬
- **Aïoli Provençal**：類似 aioli 但常加雞蛋黃；EVOO + 蒜 + 蛋黃打發成美乃滋質地
- 法式 confit（油封）：傳統用鴨油，部分現代版本以 EVOO 替代——保留耐久性
- **Beurre noisette + EVOO**：很多 fine dining 在最後 finishing 同時用兩者——奶油提供 Maillard 香、EVOO 提供 fruity 與抗氧化

---

## 西班牙（Spanish）

西班牙是世界最大 EVOO 生產國（全球 50%），**EVOO 滲透每一餐**：

- **Pan con tomate**（蕃茄麵包）：烤麵包 + 揉入半熟蕃茄 + 大量 EVOO + 海鹽；用 Picual 或 Hojiblanca 強苦辛 EVOO
- **Gazpacho**：冷蕃茄湯、EVOO 是 emulsifier 也是風味來源；安達盧西亞傳統用當地 Picual
- **Salmorejo**（Córdoba 版本 gazpacho）：濃稠版、EVOO 比例更高
- **Aioli**：純蒜 + EVOO + 鹽研磨成乳化醬（傳統不加蛋）；用溫和 EVOO（Arbequina 較適合，避免太苦）
- **Tortilla española**：用 EVOO 慢煎洋蔥與馬鈴薯（量大）→ 加蛋成圓餅；油可重複利用
- **Paella**：開頭用 EVOO 爆香 sofrito（洋蔥 + 番茄 + 大蒜）；經典 Valencia 版本油不少
- **Gambas al ajillo**（蒜炒蝦）：陶鍋 + EVOO + 大蒜 + 辣椒乾煎蝦；油本身會被當醬汁拌麵包

---

## 希臘（Greek）

- **Horiatiki**（希臘沙拉）：番茄、小黃瓜、紫洋蔥、Feta、Kalamata 橄欖 + EVOO **不加醋**（傳統純油）；用 Koroneiki
- **Horta**（野菜煮）：當季野菜水煮 + 大量 EVOO + 檸檬汁淋；地中海最簡單但最依賴油品質的菜
- **Tzatziki**：希臘優格 + 黃瓜 + 蒜 + EVOO + 蒔蘿；油在表面薄薄一層 finishing
- **Moussaka**：類似 lasagna、奶醬 + 茄子層、EVOO 煎茄子是關鍵
- **Spanakopita**（菠菜派）：菲洛酥皮層之間刷 EVOO 而非奶油（地中海齋戒傳統）
- 希臘東正教**齋期飲食**（Lent 等共 180+ 日／年禁奶蛋肉魚）：EVOO 是主要脂肪、塑造了希臘料理對 EVOO 的深度依賴

---

## 北非／中東／黎凡特

- **Hummus**：鷹嘴豆 + 芝麻醬 + 檸檬 + 蒜 + EVOO（最後表面繞圈淋）
- **Muhammara**：核桃 + 烤紅椒 + 石榴糖蜜 + EVOO；敘利亞傳統
- **Tagine**（摩洛哥燉鍋）：開頭用 EVOO 爆香香料、長時間慢燉
- **Za'atar oil**：百里香 + 漆樹 + 芝麻 + EVOO 沾醬；早餐配 pita
- **Labneh**：濾過優格 + EVOO + za'atar；類似奶油起司質地
- **Falafel**：傳統用 EVOO 油炸（中東）vs 美國版用大豆油（炸出來不同風味）

---

## 葡式（Portuguese）

- **Bacalhau à brás**：鱈魚絲 + 馬鈴薯絲 + 蛋 + EVOO（Galega 或 Cobrançosa）
- **Caldo verde**（綠湯）：羽衣甘藍湯、上桌前淋 EVOO + 加 chouriço 香腸
- **Sardinhas assadas**（烤沙丁魚）：簡單烤 + 大量 EVOO + 海鹽
