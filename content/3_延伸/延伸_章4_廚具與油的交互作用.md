# 延伸：廚具與油的交互作用

> 從 [[章4_食安與健康風險|章 4.12]] 拆出的深度版。章 4.12 留「廚具影響油氧化的概念 + 速查表」短段，這份是金屬離子催化、PTFE 熱分解、鋁鍋酸性溶出、鑄鐵 seasoning 化學的完整展開。

---

## 引言

油不是孤立加熱——它與鍋具表面有**化學交互作用**。鍋材的金屬離子、塗層、表面結構，會改變油的氧化速率、影響致癌物生成、甚至引入新的食安風險。這一節常被忽略，卻是「為什麼換鍋之後油劣化速度不同」的解釋。

---

## 1. 金屬離子催化氧化

**鐵與銅是脂質氧化的強催化劑**——它們的電子轉移能力（一價／二價、一價／三價）能加速 PUFA 雙鍵旁的氫提取、引發鏈鎖氧化。

| 鍋材 | 主要金屬離子 | 對油氧化催化作用 |
|---|---|---|
| **生鐵 / 鑄鐵（cast iron）** | Fe²⁺/Fe³⁺ | ⭐⭐⭐⭐ 強 |
| **碳鋼鍋（carbon steel wok）** | Fe²⁺/Fe³⁺ | ⭐⭐⭐⭐ 強 |
| **銅鍋（純銅 / unlined）** | Cu⁺/Cu²⁺ | ⭐⭐⭐⭐⭐ **最強**（10× Fe） |
| 銅鍋（錫襯 tin-lined） | 微弱（錫層阻隔） | ⭐⭐ 弱 |
| 不鏽鋼（304 / 316） | 微 Cr/Ni 釋出 | ⭐ 極弱 |
| 鋁鍋（純） | Al³⁺（弱催化）| ⭐⭐ 弱 |
| 陽極氧化鋁 | 表層阻隔 | ⭐ 極弱 |
| 不沾鍋（PTFE） | 無金屬接觸 | ✗ 不催化 |
| 陶瓷鍋 | 無金屬接觸 | ✗ 不催化 |
| 琺瑯（enamel） | 玻璃層阻隔 | ✗ 不催化 |

**實務意義**

- **鑄鐵與碳鋼 wok**：傳統中式爆炒主力——加速油氧化，**但中式快炒時間短（< 3 分鐘）、油劣化在容忍範圍**
- **銅鍋（unlined）**：歷史上歐洲廚房經典、現代多用於 zabaglione / confiture 等糖類加工——**避免用銅鍋長時間煎炸高 PUFA 油**（風險最大組合）；幸好現代銅鍋多為錫或不鏽鋼襯裡
- **不鏽鋼**：油氧化最少、適合長時間燉煮、煎魚與肉
- **鋁鍋 + 酸性食材**：見下節

---

## 2. 鋁離子溶出（酸性食材 + 鋁鍋）

純鋁鍋與**酸性食材**（檸檬汁、醋、紅酒、番茄）反應會釋出 Al³⁺ 離子。對油本身影響弱，但會帶入食物與人體。

- **WHO 飲水鋁建議限值**：0.2 mg/L
- **EFSA Tolerable Weekly Intake**：1 mg/kg 體重/週
- **鋁與阿茲海默症關聯**：1960-2000 年代假說、2021 後共識「無直接因果關係」（但職業性高暴露仍有疑慮）
- **解法**：用陽極氧化鋁鍋（hard anodized）或不鏽鋼鍋處理酸性料理；醃魚、燉番茄、煮葡萄醋等高酸液體避用純鋁鍋

---

## 3. 不沾鍋 PTFE / PFAS 熱分解

**PTFE（Polytetrafluoroethylene，聚四氟乙烯，俗稱 Teflon）** 是常見不沾鍋塗層。**260°C 以下穩定、超過開始分解、350-400°C 大量分解**釋放：

- **PFOA（Perfluorooctanoic Acid，全氟辛酸）**：IARC 2023 升格 **Group 1 確定致癌物**——但 2015 後主流品牌已停用 PFOA 為製造助劑（標 PFOA-Free）
- **PFOS、PFNA、新型 PFAS**：替代物、長期累積與內分泌干擾顧慮
- **PTFE 熱分解氣體**：含 perfluoroisobutylene 等急性肺毒物
- **多氟有機酸 perfluoroalkyl acids**：環境持久污染物（forever chemicals）

### 「鳥死亡現象」（Teflon flu）

養鸚鵡與小型鳥的家庭最早發現此風險——**鳥類呼吸系統對 PTFE 熱分解氣體極敏感**，過熱 Teflon 鍋會在 5-30 分鐘內導致鸚鵡死亡。**對人類則是「Polymer fume fever」**——類似流感的急性呼吸症狀，通常 24-48 小時恢復。

### 家庭使用安全範圍

| 行為 | 鍋面溫度 | 風險 |
|---|---|---|
| 一般煎蛋、煎魚 | <200°C | ✅ 安全 |
| 中火炒菜 | 200-250°C | ✅ 安全 |
| 大火爆炒（IH 全火）| 250-280°C | ⚠️ 接近上限 |
| 空鍋預熱忘記 | > 350°C | 🚨 大量分解 |
| 直火炙燒（明火）| > 400°C | 🚨 同上 |

### 結論

- 不沾鍋**正確使用安全**——中低火、不空燒、不直火
- **避免高溫煎炒、油炸、爆炒**——這些場合改用碳鋼 wok 或不鏽鋼
- **塗層刮傷後 PTFE 顆粒掉入食物**——主流毒理學認為惰性（不被吸收、隨糞便排出），但機械保護仍重要——刮傷的不沾鍋建議汰換
- 新型替代塗層（陶瓷、矽塗層、PFOA-free PFAS）耐溫差、需各別評估

## 4. 鑄鐵 seasoning 的化學

鑄鐵與碳鋼 wok 用油烘烤鍋面、形成「黑色保護層 seasoning」——**這實際上是脂質聚合反應（polymerization）**，不是「油吸進鐵裡」。

### 機制

- 油塗薄層 → 鍋面預熱 → 油的 PUFA 雙鍵被熱激活產生自由基
- 自由基連鎖反應 → **油分子聚合成 polymer network**
- 多次循環後形成耐用、疏水、不沾的硬化層
- 化學上接近「乾性油 drying oil」（如亞麻仁油、桐油塗料）的固化過程

### 選油關鍵

| 油 | 適合 seasoning？ | 原因 |
|---|---|---|
| **亞麻仁油（Flaxseed oil）** | ⭐⭐⭐⭐⭐ 最佳 | 最高 PUFA（α-亞麻酸 53%）→ 聚合最快最硬 |
| 葡萄籽油 | ⭐⭐⭐⭐ | 高 PUFA |
| 葵花油（high linoleic）| ⭐⭐⭐⭐ | 高 PUFA |
| 大豆油 | ⭐⭐⭐ | 中等 PUFA |
| 芥花油 | ⭐⭐⭐ | 中等 |
| **EVOO** | ⭐ **不適合** | MUFA 主、PUFA 少、聚合差、易黏滑 |
| 椰子油 | ⭐ 不適合 | SFA 主、不聚合 |
| 動物油 | ⭐⭐ | 早期傳統做法、但聚合不如種子油 |

**這就是為什麼專業鑄鐵粉絲推薦亞麻仁油 seasoning、但禁忌 EVOO**——化學結構決定（多酚的抗氧化保護在 seasoning 場景反而是劣勢）。

### Sheryl Canter 的經典 seasoning 配方（2010）

- 亞麻仁油 + 烤箱 250°C / 1 小時 × 6 次循環
- 形成的層比傳統豬油 seasoning 硬 5-10 倍
- 現代鑄鐵粉絲社群（Lodge Cast Iron Forum、r/castiron）的標準參考

## 5. 廚具與油的整體選用建議

| 烹飪情境 | 推薦鍋 | 推薦油 | 避免組合 |
|---|---|---|---|
| 中式爆炒（中熱）| 碳鋼 wok、鑄鐵 | EVOO、苦茶油、豬油 | 不沾鍋 + 大火 |
| 低溫煎蛋、煎魚 | 不沾鍋、不鏽鋼 | EVOO、奶油 | 鑄鐵 + 易黏食材 |
| 油炸 | 厚底不鏽鋼、鑄鐵 | 精煉橄欖、酪梨、苦茶 | 鋁鍋（酸性醃料 + 高溫） |
| 燉煮酸性料理 | 不鏽鋼、琺瑯鑄鐵 | EVOO | 鋁鍋、純銅鍋 |
| 烘焙 | 不鏽鋼烤盤、玻璃 | EVOO、奶油 | 不沾鍋（明火烘焙） |
| 高溫煎牛排（200°C+）| 鑄鐵、碳鋼 | 精煉油、奶油 finishing | 不沾鍋（過熱） |
