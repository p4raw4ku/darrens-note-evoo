# 延伸：線上購買與莊園直購策略

> 從 [[章6_市場與真偽|章 6.7]] 拆出的完整版。章 6.7 留「採購原則 5 條」短段，這份是 Amazon EU 跨境策略、莊園直購流程、警訊識別、推薦入門莊園的完整指南。

---

## 引言

**21 世紀橄欖油消費的最大變化**：高品質、可追溯的 EVOO 變得**可跨境直購**——但風險也隨之改變。下面分平台與通路討論。

---

## 跨境電商：Amazon Italy / UK / DE

### 優勢

- 直接從歐洲倉庫出貨——**收成日期最新**（通常 2-6 個月內裝瓶）
- DOP / PGI / FLOS OLEI 莊園齊全
- 價格：扣除運費後仍常比 HK 通路便宜 20-40%

### 陷阱

- **「Sold by Amazon」≠ Amazon 自有品質保證**——多數為第三方賣家（Marketplace）
- **大瓶裝重、運費可能超過油本身**
- 海運 / 空運氣候影響：夏季高溫運輸可能造成氧化
- 海關：HK 一般無關稅（自由港）；台灣 EVOO 進口稅 17%（個人 NT 3,000 以下免稅）

### 操作建議

- 選擇 **Amazon.it / Amazon.de** 官方賣家（標 *Spedito da Amazon, Venduto da Amazon*）
- 一次買 4-6 瓶分攤運費（500 ml × 4-6 瓶較佳重量）
- **冬季訂購**（11 月-3 月）——避免夏季運輸熱損
- 訂單到後第一時間檢查瓶身（無破損、密封完整、收成日期合理）

### Amazon Italy 必知用詞

| 義文 | 中譯 |
|---|---|
| Olio extravergine di oliva | 特級初榨橄欖油 |
| Raccolta | 收成（找這個關鍵字確認新鮮度） |
| Spremitura a freddo | 冷壓 |
| Filtrato / Non filtrato | 過濾 / 未過濾 |
| Spedito da Amazon | Amazon 出貨 |
| Venduto da Amazon | Amazon 售出（vs 第三方） |
| Confezione | 包裝（含瓶數） |

## 莊園直購

### 優勢

- 最新鮮（部分莊園「pre-order → 收成後直接裝瓶寄出」）
- 莊主直接溝通、可問品種、製程、blend 配方
- 部分莊園附贈品鑑卡、橄欖醬等周邊

### 推薦平台

- **Italianfood.net、Eataly Online**（義大利）
- **Aceitea de Oliva Online**、**Catacomb Olive Oil**（西班牙）
- **Quinta do Casal Novo、Esporão** 自家網站（葡萄牙）
- **Olive Japan 線上認證莊園**（日本進口 + 部分小豆島）
- **Cobram Estate、Pepper Green Estate**（澳洲、台灣均有專營）

### 注意事項

- 部分莊園**不出口亞洲、僅 EU 內部運輸**——下單前確認「shipping to Hong Kong / Taiwan」
- 運費通常較 Amazon 貴 30-50%（莊園無批量物流優勢）
- 海關文件複雜（DOP / PGI 進口需 IOC 認證文件，個人小量通常不會被卡）

## 香港 / 台灣大型線上平台

詳見 [[章5_選購與保存|章 5.5.2]] HK 通路完整地圖。重點：

- **HKTVmall**：自有品牌 + 進口品牌混雜、價格落差大；建議多比較
- **City'super E-Shop**：精選保證、價格高但品質穩定
- **PChome、momo（台灣）**：便利但收成日期常較舊（適合大瓶日常用）

### HKTVmall 採購策略

- **大促日**（雙 11、年中清、聖誕、農曆新年）：常 25-40% off
- **訂閱優惠**（部分品牌）：固定週期送貨、5-10% 折扣
- **線上獨家品牌**：HKTVmall 自有「U Mart」「Best Price」中含 EVOO 自有品牌
- **可使用門市自取**：避免運費（HK 部分配送點免費自取）

## 識別線上仿冒的警訊

| 警訊 | 解讀 |
|---|---|
| 價格顯著低於市場（例如 1L EVOO US$5）| **大概率仿冒** |
| 無收成日期、無批號 | 莊園不存在或回收瓶 |
| 評論大量「五星」但內容空泛 | review 農場操作 |
| 賣家國家與莊園聲稱地不一致 | 灰色平行進口 |
| 包裝模糊、字體偏移、印刷粗糙 | 包材偽造 |
| 「特價清倉」「過期搶手」 | 即將過期或已過期 |
| Amazon 「Sold by [陌生賣家名]」配「Ships from Italy」但 review 不到 10 條 | 詐騙概率高 |

### 進階偵測

- 用 **Google Image Search** 反查賣家照片——若大量電商共用同張產品圖、可能是貼牌
- 查賣家在其他平台（eBay、AliExpress）的同產品定價——差距 50% 以上注意
- 加入 [r/oliveoil](https://www.reddit.com/r/oliveoil/) 或 [Olive Oil Lovers Group](https://www.facebook.com/groups/oliveoillovers/) 等社群、詢問特定莊園真偽

## 直購農場興起：Fattoria / Cooperativa 模式

**Fattoria**（義大利、家族莊園）與 **Cooperativa**（西班牙、希臘合作社）近年大量開設 e-commerce 直銷：

- **採用 「subscription model」訂購**：每年新油上市自動寄送
- **限量 numbered bottle**：每瓶印序號 + 莊園地址
- **「Send pre-order to harvest」**：客戶下單後莊園當季採摘 + 裝瓶 + 立即發貨
- 透明製程：部分莊園附 GC-MS 化學分析報告與感官品鑑卡

對嚴肅消費者，這是「**避開所有零售中間環節 + 取得最新鮮可追溯 EVOO**」的最高解。但需要：

- **足夠的消費量**（一年至少 6-12 L、否則來回運費不划算）
- **語言能力**（多數莊園網站僅義 / 西 / 葡文 + 英文）
- **耐心**（pre-order 模式收貨需等 2-4 個月）

## 訂購流程實例（以 Castillo de Canena 為例）

1. **訪問** [castillodecanena.com](https://www.castillodecanena.com/)
2. **選擇產品**：Picual Family Reserve（500ml × 6 瓶禮盒）
3. **加入購物車**、輸入收貨地址（HK / TW）
4. **運費計算**：~€35-50（EU 直送亞洲、4-6 瓶）
5. **付款**：信用卡或 PayPal
6. **訂單確認**：莊園 24-48 小時內回信、附追蹤碼
7. **貨運**：DHL / FedEx Express 5-10 天到達
8. **收貨**：檢查瓶身、確認收成日期、紀錄分裝（章 5.6.5）
