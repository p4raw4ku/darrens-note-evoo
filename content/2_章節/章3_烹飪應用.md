## 3.1 油溫判斷實戰

### 3.1.1 為什麼油溫比發煙點重要

發煙點是「油開始可見冒煙」的點，**不是**「健康加熱上限」。實際上：

- 油在發煙前就已經在氧化（特別是 PUFA）
- 持續加熱在發煙點以下，傷害仍會累積
- 油溫超過 220°C 持續超過 15 分鐘，任何油都會劣化

實用觀念：**目標是「足夠熱讓食材熟、但不到油劇烈冒煙」**。多數家庭料理在 160-180°C 已完全夠用。

### 3.1.2 不用溫度計的判斷法

| 方法 | 觀察 | 對應溫度 |
|---|---|---|
| **木筷／竹筷** | 筷子周圍冒小泡 | ~150°C（低中熱）|
|  | 周圍冒密集小泡 | ~170°C（中熱）|
|  | 周圍冒大泡、滋滋響 | ~180-190°C（高熱）|
| **麵糊滴入** | 沉底慢浮 | 太冷（<140°C） |
|  | 沉底立即浮上 | 適合炸（170°C）|
|  | 浮在表面不沉 | 太熱（>200°C）|
| **油花** | 油面平靜 | 冷（<100°C）|
|  | 油面開始波動 | 中熱（150-170°C）|
|  | 油面紋路明顯、淡煙 | 高熱（190-210°C）|
|  | 濃煙 | ⚠️ **已過熱，立即降火** |

### 3.1.3 中式爆炒的真相與用油決策

**「鍋氣」常被誇大成 250°C 以上極端高溫**——實際測量推翻：

- 空鍋 30 秒可達 250°C+、但**加油下食材後實際油溫多在 160-180°C**
- 食材吸熱、釋水、被動降溫——真正持續 200°C+ 的料理少

**意義**：**家庭炒菜大多在 EVOO 完全可勝任的範圍**。所謂「鑊氣」要的是鍋面瞬間局部高溫（梅納反應、焦糖化、pyrazines 產生）、不是油本身持續 250°C。

**家用 vs 商用爐物理差異**：家用瓦斯 2.5-3.5 kW vs 商用鼓風爐 10-25 kW——餐廳鑊氣靠的是 5-10 倍家用火力 + 鑄鐵中華鍋快速熱循環。**家庭追求「足夠的梅納反應 + 食材脆度」即可，不必執著鑊氣**。

**個人化建議（華人家庭）**：

- **主力炒菜**：EVOO（一般等級、500ml ~NT 400/HKD 80）
- **強香料理**（紅燒肉、油蔥酥、麻油雞）：保留傳統豬油或麻油
- **油炸／長時間高熱**：精煉橄欖油或酪梨油
- **沙拉與淋醬**：高品質 EVOO（single estate）
- **避免日常用**：葵花、大豆、葡萄籽（PUFA 高、加熱劣化快——章 4.9）

香港、廣東與南洋廚師說的「鍋氣」（粵 *wok hei*、字面「鍋的氣息」）並非單一現象，**是四個高溫反應的疊加**：

4. **微焦化／碳化**：食材表面與鍋面接觸點達 200°C+ 的局部高溫、產生 pyrazines 與 thiophenes（核心「鑊氣」香）

**結論**：鍋氣需要的是**鍋面瞬間局部高溫（>200°C）+ 食材快速翻動接觸**，**不是**油本身持續 250°C。這也是「家用爐能不能做出鑊氣」問題的核心——關鍵在「鍋面瞬間溫度」而非「油溫」。

## 3.2 各料理選油決策

### 3.2.1 涼拌與淋醬（< 60°C）
**首選：EVOO**
- 風味、多酚、營養保留最完整
- 配對：
  - 番茄沙拉 → 強苦辛（Picual、Coratina）
  - 海鮮 carpaccio → 溫和（Arbequina、Hojiblanca）
  - 麵包沾油 → 早摘高多酚 single estate

### 3.2.2 低溫慢煮（60-120°C）
**首選：EVOO、奶油、椰子油**
- Confit（油浸慢煮）、soft scrambled eggs、低溫魚
- 慢溫不會破壞多酚

### 3.2.3 中熱煎炒（120-180°C）— **多數家庭烹飪所在**
**首選：EVOO、苦茶油、酪梨油、奶油**
- 中式日常炒青菜、炒蛋、煎魚、煎餃
- EVOO 完全勝任，多酚部分損失但 MUFA 結構穩定
- 苦茶油是台灣常見的優秀替代油

**避免**：高 PUFA 油（葵花、大豆、葡萄籽）——氧化醛類產量高

### 3.2.4 高熱煎炸（180-220°C）
**首選：精煉橄欖油、酪梨油、苦茶油（高發煙點）**
- 油炸薯條、炸雞、天婦羅、煎牛排
- EVOO 可用但風味浪費、成本高
- **避免**：奶油（150°C 即焦化）、未精煉 PUFA 油

### 3.2.5 烘焙
- **取代奶油**：可用 EVOO 但質地會改變（蛋糕更密、餅乾少酥）
- **適合甜點**：Arbequina（溫和奶油感）優於 Picual
- **餅乾／酥皮**：需固態油脂，奶油、椰子油較適合

### 3.2.6 各料理風格對照

| 料理風格 | 推薦油 | 為什麼 |
|---|---|---|
| 義式 | EVOO | 文化匹配 + 化學適合 |
| 法式 | 奶油 + EVOO | 風味為主 |
| 西班牙 | EVOO（強苦辛） | 油本身是料理一部分 |
| 中式炒菜 | EVOO 或苦茶油 | 中熱範圍兩者皆耐 |
| 中式油炸 | 精煉橄欖、苦茶、酪梨 | 高熱要求精煉 |
| 日式天婦羅 | 麻油 + 精煉植物油 | 風味平衡 |
| 印度／東南亞 | Ghee、椰子油 | 文化與風味 |
| 韓式 | 芝麻油（點綴）+ 精煉油（炒） | 風味分工 |

## 3.3 EVOO 能不能煎炒？（核心問答）

「EVOO 不能煎、發煙點低會致癌」是流傳最廣的食用油迷思。實證上 **EVOO 是測試的所有油裡最穩定的**。

### 3.3.1 Modern Olives Lab 2018 研究

ISO 17025 認證澳洲實驗室，比較 10 種市售油在加熱條件下的劣化：
- 10 種油：EVOO、virgin olive、橄欖油、芥花、米糠、葡萄籽、椰子、花生、葵花、酪梨
- 條件：開放平底鍋 250 mL 從 25°C 加熱到 240°C（20 分鐘）+ 深炸鍋 3000 mL 180°C 持續 6 小時
- 測量：極性化合物、醛類、過氧化值、酸值等

**6 小時 180°C 油炸後極性化合物**：

| 油 | 極性化合物 | 評等 |
|---|---|---|
| **EVOO** | **最低** | ⭐⭐⭐⭐⭐ |
| 酪梨油 | 低 | ⭐⭐⭐⭐ |
| 椰子油 | 低 | ⭐⭐⭐⭐ |
| 米糠油 | 15.66% | ⭐⭐⭐ |
| 芥花油 | 17.32% | ⭐⭐ |
| 葡萄籽油 | 20.24% | ⭐⭐ |
| **葵花油** | **21.75%** | ⭐ |

**結論**：EVOO 在所有測試條件下產生**最少**極性化合物與醛類；高 PUFA 精煉油劣化最快。

### 3.3.2 為什麼 EVOO 反而最穩定

三個原因疊加：
1. **MUFA 結構**（油酸 73%）：雙鍵少、不易氧化
2. **天然抗氧化物**：多酚 50-800 mg/kg、tocopherol、squalene
3. **未精煉**：保留所有保護物質

### 3.3.3 加熱對多酚的影響

- 多酚會隨加熱**部分**降解
- 但**降解產物本身仍有抗氧化活性**
- 油酸 + tocopherol 的保護力不受影響
- 結論：**EVOO 加熱後仍比未加熱的精煉油「健康」**

### 3.3.4 實務建議

| 場景 | EVOO 適用嗎？ | 備註 |
|---|---|---|
| 家庭炒菜（160-180°C，<10 分鐘）| ✅ 完全適用 | 章 3.2.3 |
| 短時間煎食材（<5 分鐘） | ✅ | 牛排、魚 |
| 烘焙（180°C） | ✅ | 注意風味影響 |
| 連續油炸（>30 分鐘） | △ 可用但風味浪費 | 用精煉橄欖實際 |
| 燒烤直接接觸火源 | ❌ | 任何油都不該用 |

### 3.3.5 EVOO 等級對烹飪用法的影響

不是「所有 EVOO 都一樣」。一支 NT 200/500ml 的超市 EVOO 與一支 NT 2,000/500ml 的單一品種早摘 EVOO，**化學組成、感官強度、抗氧化能力差距可達 10 倍以上**——用法也應該不同。下面把 EVOO 按市場區分四層、對應四種烹飪角色。

#### 等級劃分（依市場價位 + 化學品質）

| 層級 | 價位（500ml）| 多酚（mg/kg）| 收成 | 加熱適用 | finishing 適用 |
|---|---|---|---|---|---|
| **入門級 EVOO** | NT 200-500 / HKD 50-120 | 50-200 | 商業熟摘 | ⭐⭐⭐⭐ 主力 | ⭐⭐ 普通 |
| **中階 EVOO（DOP / IGP / single estate）** | NT 500-1,500 / HKD 120-400 | 200-450 | 中等成熟 | ⭐⭐⭐⭐ 適用 | ⭐⭐⭐⭐ 主力 |
| **Premium 早摘 EVOO** | NT 1,500-4,000 / HKD 400-1,000 | 450-800 | **早摘**（青橄欖）| ⭐⭐ 浪費 | ⭐⭐⭐⭐⭐ 經典 |
| **Ultra-Premium 競賽級** | NT 4,000+ / HKD 1,000+ | 700+ | 早摘 + 限量 | ❌ 浪費 | ⭐⭐⭐⭐⭐ 純品鑑 |

#### 為什麼價位影響用法

- **入門級**：精煉接近、多酚低、風味淡——**加熱時化學損失少**（本來就沒多少多酚可損失），可大量用於日常煎炒；風味淡所以不會與食材衝突；500ml 兩週用完合理
- **中階 DOP / IGP**：多酚 200-450、苦辛中等——煎炒可用但**會損失一部分多酚帶來的健康效益**；最佳是「炒菜用入門 + 起鍋淋一點中階」二元策略
- **Premium 早摘**：多酚 500+、強烈苦辛——**加熱會大量破壞風味與化學成分**；高價位用於煎炒就像「拿 NT 500 的牛肉去煮泡麵」；應該純 finishing
- **Ultra-Premium 競賽級**：純粹的品鑑與體驗、像高級酒——直接生飲、配 Carpaccio、配冰淇淋（北義傳統）；任何加熱都是浪費

#### 實務建議

**家庭兩瓶策略（推薦）**

- **主力瓶**（入門級，500-1000ml）：日常煎炒、淋醬、烘焙；2-3 個月用完
- **finishing 瓶**（中階 / Premium，250-500ml）：純淋醬、沾麵包、起鍋淋；4-6 個月用完

**三瓶策略（進階）**

- **加熱瓶**（精煉橄欖油或入門 EVOO）：高熱烹飪專用
- **日常瓶**（中階 EVOO）：中熱煎炒 + 一般沙拉
- **品鑑瓶**（Premium / Ultra）：純 finishing、配麵包、特殊場合

#### 常見錯誤

- ❌ 用 NT 3,000 的 Coratina 早摘 EVOO 煎蛋——浪費 90% 的價值（多酚加熱降解 + 強烈風味被熱料蓋過）
- ❌ 用 NT 200 的超市 EVOO 淋在 carpaccio 上——「沒風味」不是 carpaccio 的本意
- ❌ 把所有 EVOO 開封後放冰箱當「premium」呵護——除非真的吃很慢（章 5.6.4、章 7 詳述）

## 3.4 油炸的科學

油炸不只是「把食材丟進熱油」，化學上有**四個同時進行的反應**：

### 3.4.1 油炸的四種變化
1. **水解**：食材水分釋出 → 油 + 水反應 → 游離脂肪酸增加
2. **氧化**：高溫 + 空氣 → 過氧化物 → 醛類、極性化合物
3. **聚合**：自由基 → 油分子鏈連結 → 油變黏稠
4. **熱解**：極高溫 → 油碳化、起濃煙

### 3.4.2 油炸用油選擇邏輯

理想條件：
- 高 MUFA（耐氧化）
- 低 PUFA（不易劣化）
- 高發煙點（操作空間）
- 中性風味（不蓋過食材）

**最佳**：精煉橄欖油、酪梨油、苦茶油
**次佳**：精煉花生油、棕櫚油（永續爭議）、豬油（風味重）
**避免**：葡萄籽、葵花、大豆、棉籽（PUFA 太高）

### 3.4.3 重複用油的判斷

**多國法規極性化合物（Total Polar Materials, TPM）≤ 25%**：

| 國家／地區 | TPM 上限 |
|---|---|
| 台灣食藥署 | **25%** |
| EU / 法國 | 25% |
| 德國 | **24%**（最嚴）|
| 中國 | 27% |
| 香港食物安全中心（CFS） | 無明確法定上限（業界沿用國際 25% 參考） |

**實證測量**（台灣食藥署研究）：48 小時 180°C 持續炸豬排、雞胸、薯條，棕櫚油 TPM 從 11.3% 升到 **30%**（超過台灣法規）。

**家庭判斷法（不靠儀器）**：

| 信號 | 嚴重程度 | 處理 |
|---|---|---|
| 顏色變深 | ⚠️ 普通 | 觀察持續 |
| 起泡不退 | 🚨 警示 | 即將汰換 |
| 黏稠度增加 | 🚨 警示 | 同上 |
| 油耗味 | 🚨🚨 警示 | 立即汰換 |
| 焦煙提早 | 🚨🚨 警示 | 同上 |

**經驗法則**：
- 第 1-3 次：尚可（保持低溫油炸）
- 第 4-6 次：明顯劣化（適度補新油）
- 第 7+ 次：應全部汰換
- **炸肉類比炸蔬菜更傷油**（蛋白質產物加速反應；炸肉的油壽命約是炸蔬菜的 1/2-1/3）

#### 不同油的重複用油壽命對照

各油在「家用 1.5 L 油 + 180°C 油炸條件」下的合理使用次數（炸蔬菜為主）：

| 油種 | 最大使用次數 | 是否建議重複用 | 主要劣化指標 |
|---|---|---|---|
| **精煉橄欖油** | 6-8 次 | ✅ 推薦 | TPM、酸值（慢） |
| **酪梨油** | 6-8 次 | ✅ 推薦 | TPM |
| **苦茶油** | 5-7 次 | ✅ 適合 | TPM |
| **EVOO** | 5-7 次 | △ 風味浪費 | 多酚降解 + TPM |
| **棕櫚油** | 8-10 次 | △ 永續疑慮 | TPM 較慢 |
| **豬油** | 5-7 次 | ✅ 傳統做法 | 風味劣化（先於 TPM）|
| **花生油** | 5-7 次 | ✅ | TPM |
| **芥花油** | 3-5 次 | △ | 醛類快速生成 |
| **葵花／大豆／玉米／葡萄籽** | 2-3 次 | ❌ 不建議 | PUFA 氧化、醛類爆量 |

> 上表為**家庭炸蔬菜**估算；炸肉類請砍半（蛋白質殘渣加速氧化）；連續炸甜物（如甜甜圈、糖醋食材）油壽命也較短（糖焦化產物積累）。

#### 過濾舊油的方法

油炸後**趁油溫 70-80°C（不燙手但仍流動）過濾**可顯著延長使用次數：

1. **紗布 / 粗濾紙**：去除大塊麵衣、食材碎屑——基本過濾
2. **咖啡濾紙**：去除細小顆粒——可延長 1-2 次使用
3. **矽藻土（diatomaceous earth）**：商用油炸店標配、家庭可購 food-grade DE 粉、撒入熱油 15 分鐘吸附極性化合物後過濾——可延長 2-4 次
4. **馬鈴薯片過濾法（傳統）**：丟入幾片生馬鈴薯油炸至焦黑、吸附部分極性物與顏色物質——效果有限但成本低

過濾後密封儲存於陰涼處（**不入冰箱**——冰箱濕氣與低溫會加速結霜後氧化）；**最好用深色玻璃罐**（不用塑膠：高溫油接觸塑膠加速塑化劑溶出，章 4 詳述）。

#### 補油 vs 全換的時機

- **補油（每次 20-30%）**：適合「使用 2-4 次、稍微變深但無異味」的油；新油可稀釋舊油的劣化產物
- **全換**：任一「油耗味、明顯起泡、黏稠、焦煙提早」訊號出現時，立即全換、不要存僥倖

> ⚠️ **重要原則**：補油不是「無限延壽」的方法——舊油的劣化產物會像「催化劑」加速新油氧化，**整鍋油的累積使用次數仍要算（如已用 4 次、補油後再用 2 次就該全換）**。

#### 風味交叉污染

- **炸過魚 → 炸薯條**：薯條會有魚腥（不推薦）
- **炸過糖醋 → 炸鹹食**：殘留甜味 + 焦糖味（避免）
- **炸過咖哩 → 炸西式**：咖哩味會帶入後續食材
- 解法：**按味道由淡到濃排序炸**（薯條 → 雞塊 → 魚 → 帶醃料食材）；或專用油分開儲存

#### 廢油處理（重要：不能倒水槽）

- **絕對不可**：倒入廚房水槽、馬桶、戶外水溝——會凝固阻塞下水道（HK／台北每年都有大量「下水道脂肪球 fatberg」清淤事件）
- **正確做法**：
  - **台灣**：各縣市清潔隊或便利商店（部分 7-11、全家有設廢食用油回收桶）；超商收集後轉為生質柴油（biodiesel）
  - **香港**：環保署提供 *Used Cooking Oil Recycling Programme*、住宅大廈漸有回收站；HK 餐廳已強制（2014 餿水油事件後，章 4.10.4）
- **緊急家庭處理**：用舊報紙吸附後丟入一般垃圾、或裝入廢棄瓶罐丟「不可回收垃圾」——絕不直接倒

### 3.4.4 油炸減損技巧

1. **控溫穩定**：170-180°C 最佳，超過 190°C 加速劣化
2. **食材瀝乾**：水分減少水解反應
3. **預熱足**：油溫不夠會「吸油」（食材吸入過多油）
4. **一次炸量適中**：太多會降油溫、增加吸油
5. **不混新舊油**：新油被舊油「污染」加速

## 3.5 風味配對

### 3.5.1 食材 × 油的匹配
- **海鮮**：清淡 EVOO（Arbequina、Hojiblanca）；不蓋過鮮味
- **紅肉**：強烈 EVOO（Picual、Coratina）；對應油脂與蛋白質厚重
- **番茄／pasta**：草本香 EVOO（Frantoio）；地中海經典
- **沙拉**：依主食材定；綠葉 → 中性油，烤蔬菜 → 強烈
- **麵包沾油**：苦辛強的 single estate；油本身是主角

### 3.5.2 Finishing oil 用法

「最後一道工序淋上生 EVOO」是地中海料理的核心技法：
- 湯品上桌前淋
- 義大利麵拌好後淋
- 烤肉切片後淋
- 燉煮完成後淋

意義：把多酚與風味直接帶入口腔，**繞過烹飪損失**。是讓 EVOO 健康效益最大化的吃法。

### 3.5.3 調味油（Infused Oils）

蒜油、辣油、香草油的製作：
- **冷浸法**：油 + 香料密封冷藏 1-2 週
- **熱浸法**：低溫加熱 30 分鐘後冷卻、過濾

⚠️ **食安警告：肉毒桿菌（Botulism）**
- 大蒜、新鮮香草浸油 + 室溫存放 → **botulism 嚴重風險**
- 油形成厭氧環境，肉毒桿菌孢子發芽產生毒素
- **不要在室溫下保存超過幾天**
- **必須冷藏 + 1 週內用完**，或冷凍小份
- 商業浸油通常含**酸化劑**（醋酸、檸檬酸）防止
- 美國 FDA 對家庭浸油的明確警告：自製大蒜浸油是 botulism 主要食安來源之一

## 3.6 醃料降低燒烤致癌物（HCA / PAH）

燒烤紅肉產生 HCA（雜環胺）與 PAH（多環芳香烴）是高溫烹飪的主要致癌物（章 4 詳述）。**抗氧化劑豐富的醃料可顯著降低 HCA 生成**——這是有大量研究背書的有效策略。

### 3.6.1 研究發現

**Smith et al. 2008（Journal of Food Science）**——商業醃料對牛排 HCA 的影響：

| 醃料配方 | HCA 降幅 | 關鍵成分 |
|---|---|---|
| Caribbean 混合 | **-88%** | 多種多酚香料 |
| Herb 混合（迷迭香、百里香）| **-72%** | 迷迭香酸、carnosol |
| Southwest 混合 | -57% | 紅椒、小茴 |

**單一成分研究**：
- **大蒜**（20 g/100 g 醃料）：HCA **-70%**
- **迷迭香 + 百里香**：高效降幅
- 關鍵化合物：**carnosic acid、carnosol、rosmarinic acid**（迷迭香酸系）

### 3.6.2 實務醃料建議

**基本醃料公式**：
- **基底**：EVOO 1 份 + 醋／檸檬汁 1 份
- **香料**：大蒜（壓碎）、迷迭香、百里香、奧勒岡、紅椒粉
- **醃時**：紅肉至少 1 小時，最好過夜（4-24 小時）
- **效果**：可降 HCA 50-80%

**進階版**（最大化降 HCA）：
- 加入紅酒、啤酒、咖啡（含多酚）
- 加入薑黃、咖哩粉（含 curcumin）
- 加入綠茶萃取

額外好處：醃料的酸與油也讓肉質更嫩，**雙重收益**。

### 3.6.3 燒烤技法降 PAH（互補 HCA 策略）
醃料降 HCA、烹飪法降 PAH：
- 用**錫箔紙、烤盤**隔絕脂肪滴入火源
- **預先微波**牛排 / 雞胸 30 秒至 1 分鐘 → 縮短燒烤時間 → 同時降 HCA + PAH
- 改用**平底鍋、烤箱、氣炸鍋**
- 燒烤時**翻面頻繁**（每分鐘一次）
- 不吃焦黑部位

## 3.7 健康脂肪攝取量化

「該吃多少油？」是這份筆記最常被問的問題之一，但又最少有明確答案——下面把主流建議與大型試驗的實際量數據攤開比較。

### 3.7.1 主流機構建議

| 機構 | 總脂肪占熱量 | SFA | MUFA | PUFA | 反式 |
|---|---|---|---|---|---|
| **WHO 2018** | 15-30% | <10% | 主體（無上限）| 6-11% | <1% |
| **美國 Dietary Guidelines 2020-2025** | 20-35% | <10% | 主體 | 5-10% | 微量 |
| **EFSA 2010** | 20-35% | as low as possible | 主體 | 4% AI for LA | <1% |
| **台灣衛福部「每日飲食指南」2018** | 20-30% | <10% | 主體 | 6-10% | <1% |
| **Mediterranean Diet（PREDIMED 試驗版）** | **35-40%** | <10% | **20-25%（橄欖油為主）** | 5-8% | <1% |

> 注意：地中海飲食的**總脂肪比例反而高於主流建議**（35-40% vs 20-35%）——這是因為**單元不飽和脂肪酸（MUFA，橄欖油）取代了部分碳水化合物**。PREDIMED 試驗證實這個高總脂肪、低碳水的模式對心血管反而更有保護。

### 3.7.2 換算成實際量

假設一位 60 kg 的成年女性每日攝取 1,800 kcal：

| 來源 | 比例 | 熱量 | 脂肪克數 | 換算 |
|---|---|---|---|---|
| WHO 一般建議（25%）| 25% | 450 kcal | 50 g | 約 5 大匙油 |
| 台灣衛福部（25%）| 25% | 450 kcal | 50 g | 約 5 大匙 |
| 地中海飲食（35%）| 35% | 630 kcal | **70 g** | 約 **7 大匙** |

> **每大匙橄欖油約 14 g、120 kcal、含 14 g 脂肪**

### 3.7.3 大型試驗的實際攝取量

| 試驗 | 攝取量 | 觀察到的效益 |
|---|---|---|
| **PREDIMED 高劑量組** | **49.2 g/day（≈ 4 大匙）** | 心血管 -30%、糖尿病 -52% |
| **PREDIMED 中等組** | 30-40 g/day | 部分效益 |
| **Tessier 2024（最低有效門檻）** | **≥ 7 g/day（≈ 半大匙）** | 失智症死亡 -28% |
| 美國一般民眾平均 | 1-5 g/day | - |
| 義大利、希臘平均 | 30-40 g/day | - |

**結論**：

- **健康保護的最低門檻：每日 7 g（半大匙）**——Tessier 2024 失智症研究
- **顯著效益的水準：每日 30 g 以上（≈ 2-3 大匙）**——多數 cohort 研究
- **PREDIMED 試驗水準：每日 50 g（≈ 4 大匙）**——目前最強 RCT 證據
- **上限**：地中海族群有人每日達 60-80 g 仍無不良反應；無明確「上限」（但要控制總熱量）

### 3.7.4 一個典型華人家庭一日用油量

實際估算（4 口之家、3 餐自煮）：

| 用法 | 每日油量 | 每人分到 |
|---|---|---|
| 早餐（油煎蛋、菜）| 10-15 ml | ~3 ml |
| 午餐（炒 2 菜）| 20-30 ml | ~6 ml |
| 晚餐（炒 3 菜、煎 1 主菜、湯 finishing）| 40-60 ml | ~12 ml |
| 涼拌沙拉 / 麵包沾油 | 10-30 ml | ~7 ml |
| **每日總量** | **80-135 ml** | **約 20-35 g／人** |

對 4 口家庭：1 L EVOO 約 7-12 天用完——這也是為什麼**500ml 瓶常見的「兩週內用完」建議成立**。

### 3.7.5 「吃這麼多 EVOO 會胖嗎？」

最常見的擔憂。三個面向回答：

**A. PREDIMED 體重數據**
- 試驗中 EVOO 組「無熱量限制」、每日 ~50 g EVOO 攝取
- 5 年追蹤體重變化：與低脂組**無顯著差異**
- 部分研究（PREDIMED-Plus 子分析）甚至顯示腰圍下降

**B. 飽足感機制**
- MUFA + 多酚的組合**延緩胃排空、增加飽足感激素（GLP-1、CCK）分泌**
- 比同熱量碳水或 PUFA 油更不易引發暴食

**C. 替代而非添加**
- 健康吃法是**「以 EVOO 取代精煉植物油、瑪琪琳、奶油、零食加工脂肪」**
- 不是「在原本飲食上額外加 4 大匙」

> 簡單原則：**用 EVOO 取代你現在用的油**（沙拉油、葵花油、加工食品脂肪），總熱量不會大幅增加而健康指標改善；額外往原本飲食上加，才會發胖。

### 3.7.6 給日常的具體建議

**一般成人（無慢病）**：每日 20-40 g EVOO（2-3 大匙）——日常炒菜 + finishing 用法即可達成

**心血管高風險（高血壓、家族 CVD 史、代謝症候群）**：每日 40-60 g（3-4 大匙）—— 按 PREDIMED 標準

**已有 NAFLD、糖尿病前期、認知功能初顯下降**：每日 ≥ 30 g + 整體地中海飲食模式（不只是加 EVOO）

**老年人（≥ 65）**：每日 ≥ 7 g（Tessier 2024 最低門檻）——失智症保護證據

**孩童**：1 歲以上副食品可加入 EVOO；按食量比例縮減（章 7 迷思 5 詳述）

## 3.8 烹飪實用 SOP（彙整）

### 採購端
- 主力 EVOO（500-1000 ml）— 中價位、收成日期 12-18 月內
- 涼拌 EVOO（250 ml）— 高品質、single estate
- 高熱專用：精煉橄欖油、酪梨油、或苦茶油

### 烹飪端
- 中熱以下：EVOO 直接用
- 高熱：精煉油，避免持續超過 200°C
- 油炸：控溫 170-180°C、不重複超過 3 次、肉類炸過更需注意
- 燒烤：醃料含抗氧化香料、避免明火直接接觸、不吃焦黑

### 保存端
- 開封後 1-3 月用完
- 避光、避熱、避氧
- 不放冰箱（章 5、章 7 詳述）
